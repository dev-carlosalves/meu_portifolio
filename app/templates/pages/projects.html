Minha evolução em Python

Os projetos apresentados a seguir registram uma trajetória de estudos planejada e estruturada. Não foram criados de forma aleatória: cada nível representa a aquisição de uma nova competência-chave, demonstrando o crescimento prático desde os scripts iniciais de lógica até a estruturação de sistemas consolidados.

Nível 1

Fundamentos da Programação

Nível 1 Concluído

Conversor de Temperaturas

Utilitário de terminal projetado para realizar conversões precisas entre as escalas termométricas Celsius, Fahrenheit e Kelvin.

Competências Desenvolvidas:

Entrada/saída de dados, manipulação de variáveis, operadores matemáticos e controle de fluxo básico.

Python Lógica
Nível 1 Concluído

Calculadora de IMC

Script interativo que solicita peso e altura do usuário, calcula o IMC e classifica a faixa de peso de acordo com os parâmetros da OMS.

Competências Desenvolvidas:

Uso prático de blocos condicionais encadeados (if-elif-else) e formatação de casas decimais.

Python Matemática
Nível 1 Concluído

Jogo de Adivinhação

Jogo interativo no terminal no qual o computador escolhe um número aleatório e o usuário tem tentativas para acertar, recebendo dicas se o valor é maior ou menor.

Competências Desenvolvidas:

Geração de números aleatórios, loops condicionais controlados por flags e validação de erros simples.

Python Terminal

Python Listas

Python Strings Estatística
